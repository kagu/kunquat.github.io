Installing

To use Kunquat, you need to get your hands on a relatively new (we use 12.04) Ubuntu system. You can then download and install kunquat with the following command line magic.

sudo -k apt-get install build-essential scons libwavpack-dev libpulse-dev python-qt4 check libsndfile1-dev wget
cd /tmp
wget http://kunquat.com/releases/kunquat-0.6.1.tar.gz
wget http://kunquat.com/releases/kunquat-0.6.1.tar.gz.md5
md5sum -c kunquat-0.6.1.tar.gz.md5
tar xzf kunquat-0.6.1.tar.gz
cd kunquat-0.6.1
scons
scons prefix=~/kunquat install

Running

At this point you should have Kunquat installed. You can start the Kunquat tracker as follows.

export LD_LIBRARY_PATH=~/kunquat/lib
export PYTHONPATH=~/kunquat/lib/python2.7/site-packages
~/kunquat/bin/kunquat-tracker
